# Library-Management-System
## Overview
Web application for library operations: book management, user registration, and borrowing records.

## Tech Stack
- Frontend: React.js, CSS
- Backend: Node.js, Express.js
- Database: MySQL
- APIs: RESTful

## Features
- Search Books
- Login / Register
- Role based Dashboard (Librarian/ Student)
- Reserve/ Borrow Books
- Add/ Delete Books

## Setup
1. Clone the repository: https://github.com/anjali2626/Library-Management-System                                                      
2. Install dependencies: npm install
3. Configure database connection in .env
4. Run client_side: npm start
5. Run backend: node server.js

## Usage
- Librarians manages books & students.
- Students borrow/return/search books.
